Especificidad de los selectores

Entre mayor especificidad, mayor va a ser la probabilidad de que sus declaraciones se usen sobre las demas.

1- X 0 0 0 0 | !Important

2 - X 0 0 0 | Estilos en linea

3 - X 0 0 | #ID

4 - X 0 | Clases, atributos y pseudoclases

5 - X | Elementos y pseudoelementos

6 - 0 | Selector universal
